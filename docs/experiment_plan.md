# 实验方案与消融实验设计
项目：LLM Multi-Agent MAPPO + Self-Play 多智能体强化学习系统
本文档统一规划所有对照实验、消融实验、性能测试，用于验证算法模块有效性、工程优化收益，是项目量化成果、面试答辩、图表展示的核心依据。
所有实验严格遵循单一变量控制原则，保证结论可信。

## 一、实验总体说明
### 1.1 实验目的
1. 验证多智能体MAPPO相比单智能体PPO的任务完成效果提升
2. 验证自博弈迭代机制对长流程复杂任务收敛的增益
3. 验证分层奖励函数各层级对训练稳定性、任务成功率的贡献
4. 验证分布式训练、Rust沙箱、低延迟推理模块的工程性能优化收益

### 1.2 统一实验环境
- 基础大模型：Llama3-8B
- 训练硬件：单机4卡GPU
- 训练底座：PyTorch FSDP半精度混合训练
- 交互环境：自研多智能体工具调用任务数据集
- 评价指标：任务完成率、平均回报、收敛所需步数、推理延迟、每秒训练样本吞吐

## 二、实验一：单智能体PPO与多智能体MAPPO对照消融实验
### 2.1 控制变量
仅修改是否启用全局集中式价值网络，其余所有参数完全固定。
### 2.2 分组设置
对照组：独立单智能体PPO，每个智能体仅使用自身局部观测、局部奖励更新策略
实验组：MAPPO多智能体训练，全局统一价值基线，所有智能体朝着整体任务目标优化
### 2.3 预期结论
MAPPO依靠全局价值网络解决多智能体各自利己、协作失效的问题；
规划、编码、校验三类智能体分工配合，长流程复合任务完成率明显高于独立单智能体方案。
### 2.4 产出可视化内容
任务完成率收敛折线图、全程平均回报对比图

## 三、实验二：关闭/开启自博弈机制对照实验
### 3.1 控制变量
仅控制是否加载历史模型参与交互采样，统一使用MAPPO训练逻辑、奖励配置。
### 3.2 分组设置
对照组：仅使用随机初始化固定智能体交互，无历史旧模型参与采样
实验组：自博弈循环迭代，历史版本与当前主模型组队交互，自动生成高难度样本
### 3.3 预期结论
自博弈机制无需人工大批量标注复杂任务样本；随迭代轮次增加，交互任务难度自动提升，模型对长尾复杂需求泛化能力更强，推理阶段错误率明显下降。
### 3.4 产出可视化内容
有无自博弈机制下的收敛曲线、任务错误率对比图

## 四、实验三：分层奖励函数消融对照实验
### 4.1 控制变量
调整奖励组合层级，其余训练参数、交互环境完全一致。
三层奖励分别为：任务终局完成奖励、工具执行规范奖励、多智能体协作配合奖励。
### 4.2 三组实验组
实验组1：仅使用任务完成终局奖励
实验组2：终局奖励 + 工具执行规范奖励
实验组3：完整三层奖励叠加（最终项目使用方案）
### 4.3 预期结论
仅终局奖励训练信号稀疏，模型收敛缓慢、震荡明显；
增加工具执行奖励后，智能体调用工具格式规范度大幅提升；
完整三层奖励组合收敛速度最快，多智能体配合效果最优，最终任务成功率最高。
### 4.4 产出可视化内容
不同奖励组合的收敛速度、最终任务完成率对比柱状图

## 五、实验四：单卡训练与4卡FSDP分布式训练性能对照
### 5.1 控制变量
仅切换单卡/4卡分布式训练模式，模型、数据集、超参保持不变。
### 5.2 分组设置
对照组：单卡完整模型训练，无参数分片、无多卡通信优化
实验组：4卡FSDP分片训练，搭配激活重计算、半精度显存优化
### 5.3 对比性能指标
单卡/4卡显存占用峰值、每秒训练样本吞吐、单轮完整迭代耗时、超大批次训练稳定性
### 5.4 预期结论
FSDP参数分片机制大幅降低单张显卡显存占用，可稳定运行8B规模大模型；
多卡并行成倍提升训练吞吐，搭配通信计算重叠优化，大幅缩短完整训练周期。
### 5.5 产出可视化内容
多卡训练吞吐提升柱状图、单卡/4卡显存占用对比表格

## 六、工程配套性能测试
1. Rust沙箱性能测试：原生Python代码执行与Rust异步批量执行对比，指标包含单任务延迟、批量并发吞吐量、长时间运行稳定性
2. vLLM推理性能测试：原生HuggingFace推理与分页缓存低延迟推理对比，指标包含首字生成延迟、每秒并发请求承载量

## 七、整体实验总结
项目全部算法、工程优化都通过单一变量消融实验验证有效性：
1. MAPPO多智能体架构解决多角色协作冲突，提升复杂任务完成率
2. 自博弈迭代机制降低人工标注成本，持续提升模型泛化能力
3. 三层分层奖励缓解稀疏奖励训练难题，强化智能体分工配合
4. FSDP分布式底座解决大模型显存瓶颈，提升整体训练吞吐
5. Rust沙箱、低延迟推理服务解决仿真、线上部署性能短板
